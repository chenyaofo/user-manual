# 介绍

SMIL实验室目前共有可用GPU服务器25台，纯CPU服务器1台。共有GPU卡158块，CPU线程数896

SMIL Infrastructure团队负责为同学们提供服务器使用过程中的技术支持，服务器软件维护升级，以及联系服务器硬件的维修等。
